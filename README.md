## 📚 Cos'è questo progetto

Questo repository contiene il **lettore web interattivo** del libro digitale "La Rivoluzione del Quando". Non è un semplice PDF: è un'esperienza di lettura moderna con:

- 🎯 **Indice laterale navigabile** con progress bar globale
- ✅ **Tracciamento dei progressi** (capitoli letti salvati in localStorage)
- 📱 **Design responsive** (funziona su desktop, tablet e mobile)
- ⌨️ **Scorciatoie da tastiera** (frecce ← → per navigare)
- 🎨 **Design coerente** con il brand del prodotto

## 📁 Struttura del progetto

### Descrizione dei file

| File | Scopo | Da modificare quando... |
|---|---|---|
| `index.html` | Struttura HTML del lettore | Cambi il layout generale |
| `book-styles.css` | Stili e design | Vuoi cambiare colori, font, layout |
| `book-script.js` | Logica JavaScript | Modifichi il comportamento del lettore |
| `chapters.js` | **Contenuto dei capitoli** | **Aggiungi o modifichi capitoli** |

## 🚀 Come usarlo

### Per l'utente finale

1. Apri il link fornito dopo l'acquisto (es. `https://libro-rivoluzione.vercel.app`)
2. Usa l'indice laterale per navigare tra i capitoli
3. Clicca su **"✓ Segna come letto"** per tracciare i tuoi progressi
4. Usa le frecce ← → della tastiera per navigare velocemente
